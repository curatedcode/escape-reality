# Escape Reality Website

This is the marketing website for **Escape Reality**, a escape room business.

Built and designed by Zackary Fotheringham for the client. I do not own or operate Escape Reality.

## Tech Stack
- **Framework:** Astro + React + AlpineJS
- **Styling:** Tailwind CSS
- **Deployment:** AWS

## Features
- Responsive layout
- SEO and social metadata
- Client-managed content and images
- Simple static build for speed and reliability
